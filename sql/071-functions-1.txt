 Write a query to display all results for all employees in
        lowercase. Order the reults by name (alphabetically) and job
        (alphabetically).

            Name       Job          Salary
            ---------- --------- ---------
            blake      manager        2850
            ...
            king       president      5000
            ...


Select lower(ename) as "Name", lower(job) as "Job", sal as "Salary"
from emp
order by ename asc;
